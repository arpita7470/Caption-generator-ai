# Caption Generator AI

## Setup Instructions
1. Clone the repository:  
   ```bash
   git clone https://github.com/arpita7470/Caption-generator-ai.git
   cd Caption-generator-ai
   ```
2. Install dependencies:  
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:  
   ```bash
   python app.py
   ```

## Features
- Generate captions for images using AI.
- Supports various image formats.
- User-friendly interface.
- Customization options for captions.
